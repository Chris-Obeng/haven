Build a premium, mobile-first homepage for "Haven Boutique Hotel" — a luxury boutique 
hotel in Kumasi, Ghana's Ashanti Region. The design must match the editorial luxury 
aesthetic of hba.com exactly: warm cream/off-white background, deep forest green 
accent sections, large serif headings, minimal navigation, and cinematic full-bleed 
imagery with parallax scroll effects.

---

## TECH STACK
- Next.js App Router + TypeScript
- Tailwind CSS for layout/spacing
- GSAP (with ScrollTrigger) for all scroll animations and parallax
- Framer Motion for micro-interactions (hover states, button transitions)
- next/image for optimized images
- next/font for Google Fonts

---

## TYPOGRAPHY (critical — must match exactly)
- Headings: "Cormorant Garamond" from Google Fonts — weights 300, 400, 500
  (font-display: swap, italic variant for pull quotes)

  letter-spacing 0.25em, ALL CAPS, font-weight 400

## ANIMATIONS (GSAP — implement all of these)
1. Hero parallax: image translateY at 40% scroll speed via ScrollTrigger
2. Section image fade-up: every editorial image fades from opacity:0 + y:60 → normal
   on scroll entry, stagger 0.15s between multiple images
3. Staggered gallery parallax: left/right images move at different speeds (0.3x vs 0.5x)
4. Text reveal: section headings clip-reveal from bottom using GSAP + clip-path animation
5. Dark sections: fade + slide-up 40px on scroll entry
6. Quote word-by-word reveal: split text, stagger each word 0.04s opacity reveal
7. Nav background transition: transparent → cream smooth transition on scroll past 80px
8. Room cards: hover scale 1.03 on image, Framer Motion whileHover
9. CTA buttons: Framer Motion spring scale on hover + subtle border glow

## MOBILE-FIRST RESPONSIVE RULES
- Base styles written for 375px (iPhone SE)
- All font sizes use clamp() for fluid scaling
- Hero text: max 3 lines on mobile, reduce letter-spacing
- Two-column layouts: stack to single column below 768px
- Horizontal room card scroll on mobile (snap scroll, -webkit-overflow-scrolling: touch)
- Navigation overlay: full screen on mobile with large tap targets (min 48px)
- Images: aspect-ratio locked, never overflow on small screens
- Padding: 24px horizontal on mobile, 80px on desktop sections
- Footer: 1-column stack on mobile

## CODE QUALITY REQUIREMENTS
- All components in /components folder with TypeScript interfaces
- GSAP initialized in useEffect with proper cleanup (ScrollTrigger.kill())
- next/image with proper width/height and sizes prop for responsive
- No layout shift: reserve image dimensions with aspect-ratio CSS
- Semantic HTML: <section>, <header>, <nav>, <main>, <footer>, <article>
- aria-labels on all icon buttons and navigation
- Smooth scrolling: html { scroll-behavior: smooth }
- CSS custom properties for all brand colors defined in :root
